# 🐍 Snake Game - Python Turtle

A classic Snake game built entirely with Python Turtle library,
demonstrating real-time event handling, collision detection,
and game loop logic.

## 🎯 What It Does
- Real-time snake movement with keyboard controls
- Collision detection (walls and self)
- Score tracking system
- Progressively increases difficulty as snake grows

## 🛠️ Technologies Used
- Python
- Python Turtle library
- Event-driven programming
- Object-oriented logic

## 📁 Files In This Repository
- `main.py` — Main game code and logic
-  `food.py` — The object that causes the growth of the snake
-  `scoreboard.py` — Accumulation of food eaten
-  `bodyofsnake.py` — The snake shown in the screen
- Screenshots — Gameplay visuals
